# 🎬 Movie Rating Analyzer using Python

The **Movie Rating Analyzer** is an interactive data analytics project developed using **Python**, **Pandas**, **Matplotlib**, and **Streamlit**. It analyzes movie ratings from the **MovieLens** dataset and presents useful insights through an easy-to-use web dashboard. Users can explore movies, visualize trends, search titles, apply filters, and discover recommendations—all from a single interface.

## 📌 Project Overview

This project was created to demonstrate the complete data analysis workflow, from raw data to an interactive dashboard.

The project includes:

* Importing and exploring the MovieLens dataset
* Cleaning and preparing the data for analysis
* Performing Exploratory Data Analysis (EDA)
* Generating meaningful insights using visualizations
* Building an interactive dashboard with Streamlit
* Deploying the application online for public access

The dashboard allows users to:

* 🔍 Search movies by title
* 🎭 Filter movies by genre
* 📅 Filter movies by release year
* 📊 Explore different charts and visualizations
* ⭐ View movie recommendations based on genre and ratings

This project helped strengthen my understanding of data analysis, data visualization, dashboard development, and deploying Python applications. It also demonstrates practical use of **Pandas**, **Matplotlib**, and **Streamlit** in building a real-world analytics application.
